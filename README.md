# ✈️ Airline Reservation System

A full stack web application built using:

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Python Flask

Database:
- MySQL

## Features
- User Registration
- User Login
- Flight Search
- Ticket Booking
- Food Ordering
- Crew Information

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Import database in MySQL Workbench

3. Run the application
python app.py

4. Open browser
http://127.0.0.1:5000
